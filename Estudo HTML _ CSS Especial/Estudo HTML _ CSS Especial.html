Animações


Animações em CSS3


Em animações em CSS são possibilidades de ir de uma formatação específica para outra formação.


Para usar animação precisamos usar o "@keyframe" na folha de estilo do projeto


Exemplo:

@keyframe "nome da animação" {
from {}
to{}
}


No objeto da animação devemos colocar:


animation-name: "nome da animação" ;

animation-duration: 1s ;

animation-delay: 1s ;

animation-iteration-count: 10 ;

animation-direction: alternate;


OBS:. verificar os pré fixos dos Browsers


Transições


Usando Transições


Para usarmos transições precisamos utilizar os estados de cada elemento ( hover, e etc )

Desta forma, definimos atributos para o elemento em estado padrão, e logo após, os atributos nos estados definidos.


Dentro dos estados definido ( Ex. Hover ), incluimos o atributo " transition: all 3s; " ou podemos definir quais propriedades farão parte da transição.

Flexbox - No Container


Novo modo de Layout, para organizar os elementos dentro do Container de forma mais simples.


Atributo No Container = " display: flex ; "


" flex-direction: row-reverse ; " ou " row / column / row-reserve / column-reverse "

Alinha os itens em Colunas ou em Linhas


" flex-wrap: nowrap" ou " wrap "

Define se os itens vão quebrar para a próxima linha ou não.


" justify-content: flex-end " ou " flex-start / flex-end / center / space-between / space-around "

Ajusta os itens ( na horizontal ) de acordo com o espaçamento do Container


" align-itens: center " ou " stretch / center / flex-start / flex-end / baseline "

Ajusta os itens ( na vertical ) de acordo com o espaçamento do Conteiner



Flexbox - Nos Itens


Atributo No Container = " display: flex ; "


Define uma classe "ordem" para os itens e dentro a classe define o atributo
" order: nº da ordem ; "

OBS.: As ordens dos itens iniciam em 0 ( zero )


" flex-grow: 0 ; " = Estica o item até preencher o container


" flex-shrink: 1 ; " = Encolhe o item proporcionalmente conforme o tamanho do container


" flex-basis: auto ; " = Indica o tamanho inicial do item antes da distribuição do espaço restante no container
